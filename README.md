# Moonshot-App - Swift UI

The app displays information from [Apollo crewed missions](https://en.wikipedia.org/wiki/List_of_Apollo_missions).  
### Major Concepts:
* Codeable, Identifable
* Bundle decoder
* Extensions
* Generic Types
* Geometery Reader
